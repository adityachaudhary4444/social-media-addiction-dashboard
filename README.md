# Social Media Addiction Analysis

## Project Overview
This project investigates the growing phenomenon of social media addiction among students, exploring how digital habits affect mental health, academic performance, and daily life. By analyzing data on student behavior, platform usage, and lifestyle patterns, the project identifies key indicators of addiction and its broader impact.

## Key Features & Insights
- **Behavioral Analysis:** Examines average daily usage, preferred platforms, and usage patterns.  
- **Academic Correlation:** Evaluates the relationship between social media usage and academic performance.  
- **Mental Health & Lifestyle Impact:** Studies sleep patterns, stress levels, and interpersonal conflicts caused by excessive social media use.  
- **Addiction Scoring:** Develops a comprehensive addiction score to quantify the risk of social media dependence.  
- **Actionable Recommendations:** Provides insights for students, educators, and mental health professionals to mitigate negative effects of social media overuse.

## Technologies & Tools
- **Data Analysis:** Excel  
- **Visualization:** Power BI (DAX measures, calculated columns, interactive dashboards)  
- **Techniques:** Data cleaning, exploratory data analysis (EDA), statistical correlation, and scoring algorithms
